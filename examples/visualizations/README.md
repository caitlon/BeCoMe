# BeCoMe Visualizations

This directory contains visualization scripts for the BeCoMe (Best Compromise Method) algorithm.

## Files

- `visualize_become.py` - Interactive Jupyter notebook with comprehensive visualizations for BeCoMe results

## Visualizations Included

1. **Triangular Membership Functions** - Display of expert opinions as fuzzy triangular numbers
2. **Centroid Charts** - Comparison of expert centroids with aggregated metrics
3. **Range Comparison Heatmaps** - Visual representation of opinion ranges (lower-peak-upper)
4. **Interactive Sensitivity Analysis** - Dynamic exploration of expert inclusion/exclusion impact
5. **Scenario Dashboard** - Comparative overview of all three case studies
6. **Accuracy Gauge Indicators** - Visual quality assessment of expert agreement

## Usage

Open `visualize_become.py` in Jupyter Notebook or JupyterLab:

```bash
jupyter notebook visualize_become.py
```

Or use JupyterLab:

```bash
jupyter lab visualize_become.py
```

## Output

All visualizations are saved to the `output/` directory within this folder (`examples/visualizations/output/`).
