# BeCoMe Method Implementation

Python implementation of the BeCoMe (Best Compromise Mean) method for group decision-making support under fuzzy uncertainty.

## Project Status

This project is under active development.

## Overview

BeCoMe is a method for aggregating expert opinions expressed as fuzzy triangular numbers to find the optimal compromise solution. The method was developed at the Czech University of Life Sciences Prague.

## Installation

```bash
python -m venv venv
source venv/bin/activate  # On macOS/Linux
pip install -r requirements.txt
```
